# ГIалгIай мотт — Проверка орфографии ингушского языка

Веб-приложение для проверки правописания текстов на ингушском языке. Совмещает словарную проверку с контекстным анализом на основе N-грамм.

## Технологии

| Слой | Стек |
|---|---|
| Backend | Spring Boot 4.0.3, Java 25 |
| Frontend | React 19, Vite 7 |
| NLP | Расстояние Левенштейна, биграммная языковая модель |
| Данные | PDF-словари (Куркиев, Барахоева, Кодзоев, Мерешков, Оздоева, Тариева), корпус «Лоаман Іуйре» |

## Запуск

**Backend** (порт 8080):
```bash
./gradlew bootRun
```

**Frontend** (порт 5173):
```bash
cd frontend
npm run dev
```

Открыть: http://localhost:5173

## Как работает проверка

**Орфография:**
1. Текст нормализуется — все варианты ингушской «палочки» (ӏ, Ӏ, цифра 1, латинская I) приводятся к единому символу
2. Каждое слово ищется в словаре
3. Для слов не из словаря подбираются варианты с расстоянием Левенштейна ≤ 2
4. Каждый вариант дополняется переводом из `ingush_translations.json`

**Контекст:**
1. Биграммная модель оценивает вероятность пар соседних слов
2. Статистически невозможные сочетания помечаются жёлтым предупреждением

**UI:**
- Ошибки подсвечиваются красным, контекстные предупреждения — жёлтым
- Клик на слово открывает попап с вариантами и их переводами на русский
- Выбор варианта заменяет ошибочное слово в тексте по позиции (корректно с кириллицей)
- Вкладка «Словарь» — поиск по 112 000+ словам с пагинацией
- Дебаунс 800 мс — проверка запускается после паузы в наборе

## Морфология ингушского языка

### Источники

| Источник | Тип | Что даёт |
|----------|-----|----------|
| Johanna Nichols, «Ingush Grammar» (2011, UC Press, 829 с.) | Академическая грамматика (EN) | Полное описание системы падежей, словообразования, спряжения |
| «Словообразование и словоизменение в нахских языках» (1986) | Сборник статей | Деривационные суффиксы, именное и глагольное словоизменение |
| «Вопросы лексики и словообразования вайнахских языков» (1987) | Сборник статей | Лексикология, продуктивные словообразовательные модели |
| «Вопросы вайнахского синтаксиса» (1982) | Сборник статей | Синтаксис, управление, именные группы |
| Куркиев А.С., «Основные вопросы лексикологии» (1979) | Монография | Пласты лексики, заимствования, словообразование |

### Система падежей (8 падежей, по Nichols 2011)

| Падеж | Суффикс(ы) | Функция |
|-------|-----------|---------|
| Именительный | — | подлежащее интранзитивного глагола, пациенс |
| Родительный | -а, -ан, -н | атрибутивный генитив |
| Дательный | -аа, -ана, -нна | реципиент, бенефициант |
| Эргативный | -уо, -аз | агенс транзитивного глагола |
| Аллатив | -ага, -га, -нга | направление движения |
| Инструментальный | -аца, -ца, -нца | инструмент, способ |
| Латив | -агI | направление (вариант аллатива) |
| Сравнительный | -ал, -нал, -л | стандарт сравнения |

Множественное число: **-аш** (продуктивный), **-ий** (одушевлённые), **-ой** (с суффиксом -хьуо), **-рч** (животные).

### Словообразование существительных (ключевые суффиксы)

| Суффикс | Значение | Примеры |
|---------|---------|---------|
| -ам | девербальный, результат | боахам (имущество), бустам (мера), лотхIам (жалоба) |
| -ар | девербальный, процесс/действие | да'ар (еда), хьехар (учение) |
| -хьуо | агентивный («тот, кто делает») | лоалахьуо (сосед), дешархьуо (ученик) |
| -джг | диминутив/уменьшительное | жIалиг (щенок), бераг (ребёночек) |

### Ключевая сложность — аблаут

В косвенных падежах **корневой гласный меняется** непредсказуемо:
- `бер` (ребёнок) → `биера` (ген.), `биерага` (алл.)
- `куор` (окно) → `куора` (ген.)
- `бордз` (волк) → `берза` (ген.)
- `мах` (цена) → `меаха` (ген.)

Аблаут не выводится правилами — нужен отдельный словарь основ.

### Что реализовано

- [x] **Нормализация палочки** — все варианты ӏ (U+04CF, U+04C0, цифра 1, латинская I) приводятся к единому символу
- [x] **Стеммер `IngushStemmer`** — 32 суффикса, упорядоченных по длине (longest match first):
  - Падежи мн.ч.: `-ашта`, `-ашка`, `-аьшта`, `-аьшка`, `-аьшца`
  - Мн.ч. именительный: `-аш`, `-аьш`, `-ий`, `-ой`
  - Падежи ед.ч.: `-ага`, `-аца`, `-гӏа`, `-хьа`, `-ара`, `-ача`, `-нна`, `-ана`, `-нах`
  - Краткие: `-га`, `-ца`, `-ал`, `-ах`, `-на`, `-хь`, `-ан`, `-аа`, `-л`, `-й`
  - Покрытие: ~70% регулярных флектированных форм (без аблаута)

### Что можно сделать дальше

**Аблаутные чередования** (высокий приоритет)
- Извлечь пары основа→косвенная форма из словаря Куркиева (там приводятся основы)
- Например: `бер`→`биера`, `куор`→`куора`, `бордз`→`берза`
- Хранить как отдельный JSON-файл `ingush_ablaut.json`
- Покроет дополнительно ~15-20% флектированных форм

**Глагольная морфология**
- Ингушский глагол имеет классовые префиксы (б-, й-, в-, д-) согласующиеся с абсолютивом
- Временны́е/видовые суффиксы: простое прошедшее, перфект, прогрессив, конверб, масдар
- Источник: Nichols гл. 12–13; потребует отдельного глагольного стеммера

**Полный морфологический анализатор**
- Разбор слова на морфемы с указанием части речи, падежа, числа
- Позволит не просто проверять орфографию, но и грамматику (согласование)
- Трудоёмко, но реально при наличии полного парадигматического словаря

**Словарь основ из Куркиева**
- В словаре Куркиева после каждого слова указаны косвенная основа и грамматический класс
- Можно извлечь скриптом и создать `ingush_stems.json`: `{"бер": "биера", "куор": "куора", ...}`
